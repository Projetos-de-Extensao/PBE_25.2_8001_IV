## Como Usar

1. **Instalar dependências**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Executar servidor local**:
   ```bash
   mkdocs serve
   ```

3. **Build para produção**:
   ```bash
   mkdocs build
   ```
